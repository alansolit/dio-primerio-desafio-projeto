#### Comandos Básicos Git 

- **ls** - Listar conteúdos em uma pasta ou diretório.
- **cd** - Ele é utilizado para a navegação entre as pastas do computador e diretórios.
- **cd ..** - Volta na pasta ou diretório anterior.
- **clear ou ctrl+L** - para limpar a linha de comando.
- **git init** - cria o repositório do git na pasta escolhida.
- **git status** - verifica qual status do arquivo Untracked, Modfied ou Stage.
- **git add * **-  adciona arquivos untracked para stage.
- **git commit -m "descrição"** - Processo de commit.
- **git push origin main ou master** - envia o conteúdo do repositório local para um repositório remoto.
