#### Comandos Básicos CMD

- **dir** - com esse comando é possível listar o conteúdo contido em um diretório.
- **cd** - Ele é utilizado para a navegação entre as pastas do computador e diretórios.
- **cd..** -Volta na pasta ou diretório anterior.
- **mkdir** - Comando para criar uma pasta.
- **del  "nome da pasta"** - Comando para excluir arquivos de uma pasta.
- **rmdir  "nome da pasta" /S /Q** - Comando para excluir pasta ou repositório local.
- **tab** - Função auto-completar.
- **cls** -Limpa a tela de comando.
